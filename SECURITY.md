# Security and Data Handling

Please report credential exposure, unsafe file access, hidden-data leakage, or
release-boundary violations privately to `colinwang@gatech.edu`.

Do not open a public issue containing:

- API keys, private keys, tokens, or provider credentials;
- licensed Databento row-level records;
- raw model-provider responses;
- community-hidden outcomes or decrypted holdout proposals; or
- private institutional authorization data.

FinAuth-Audit is a research benchmark. It is not investment advice, a live
trading system, or authorization for deployment.
